# External Integration

```js
mplayer("send", action, payload);

// Hide Controls
mplayer("send", "hideControls");
// Show Controls
mplayer("send", "showControls");
```

| Function          | action       | payload |
| ------------- | ------------ | ------- |
| Hide Controls | hideControls | N/A    |
| Show Controls | showControls | N/A   |
